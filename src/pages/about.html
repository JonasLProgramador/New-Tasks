Sobre o Gerenciador de Tarefas

O Projeto

O Gerenciador de Tarefas é uma aplicação web simples desenvolvida para ajudar os usuários a gerenciar suas tarefas diárias de forma eficiente. Utilizando jQuery para manipulação do DOM e AJAX, a aplicação se integra com a API do Todoist para fornecer funcionalidades robustas de gerenciamento de tarefas.

Funcionalidades

  • Autenticação com Todoist usando OAuth 2.0.
  • Adicionar novas tarefas ao Todoist.
  • Listar todas as tarefas existentes no Todoist.
  • Deletar tarefas do Todoist.

Tecnologias Utilizadas

O projeto utiliza as seguintes tecnologias:

  • jQuery: Para manipulação de DOM e AJAX.
  • Todoist API: Para gerenciar tarefas.
  • HTML5 e CSS3: Para a estrutura e estilo da aplicação.
